How it works

The state lives in the 2D plane spanned by \(\ket{B}\) (bad states) and \(\ket{G}\) (good states). Each Grover iteration applies two reflections: the oracle reflects through \(\ket{B}\), then the diffusion operator reflects through \(\ket{U}\). The net effect rotates the state by \(2\theta\) towards \(\ket{G}\).
